# Sanchi Foundation & Sanchaya — Annual Report 2022–2025

A co-branded, single-page annual report designed for funders, partners, and supporters.

## 📁 Files

| File | Purpose |
|------|---------|
| `index.html` | The complete annual report — self-contained, no build step needed |
| `README.md` | This file |

## ✅ Features

- Co-branded with both Sanchi Foundation (red) and Sanchaya (purple/magenta) identities
- All 126 video links clickable, linking directly to YouTube
- All 30+ Sanchaya portals linked
- Donation section with links to both support pages
- Fully responsive (mobile-friendly)
- Fixed navigation with smooth scroll
- No dependencies — works offline after first load (Google Fonts cached)
